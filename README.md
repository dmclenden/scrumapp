 # Scrum Self-Study
<br>
A website for people that are studying for the Professional Scrum Master I exam.
<br>
You can register, find recources to help you study, and add and delete notes.
<br>
<br>


### Used:
<br>
* Node
<br>
* Express for backend routes
<br>
* Postgres database
<br>
* Sequelize for ORM
<br>
* Mustache templating
<br>
* Bootstrap
